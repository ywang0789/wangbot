# Just an innocent discord bot

## Installation
Requires python 3.7+

### 1. Create your own './secret/keys.json'

#### keys.json file structure

```
{
    "gpt_api_key": "asbasd",
    "discord_token": "asdasdasd"
}

```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run

```
python main.py
```
