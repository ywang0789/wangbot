# Just an innocent discord bot

## Installation
Requires python 3.7+

### 1. Create your own './secret/keys.py'

#### keys.py file structure

```

OPENAI_API_KEY = asdf
OPENAI_API_KEY = xczbv


```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run

```
python main.py
```
